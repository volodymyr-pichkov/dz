# Express Server with Favicon, Cookies, and JWT Authentication

## Структура

- **/public/favicon.ico**: Файл иконки для сайта.
- **/views/index.pug**: Шаблон для PUG.
- **app.js**: Серверный код с функциональностью для favicon, cookies и JWT.

## Функциональность

Сервер обслуживает favicon на всех страницах.

Пользователи могут выбрать тему оформления, которая сохраняется через cookies.

Регистрация пользователей с генерацией JWT.

Доступ к защищённым маршрутам с проверкой JWT.

## Установка

1. Клонируйте репозиторий.
2. Установите зависимости:
   npm install
